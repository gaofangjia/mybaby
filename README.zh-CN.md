# 儿童成长表现记录 (Android App)

一款高颜值、充满温馨治愈卡通设计风格的 **儿童成长表现记录** 安卓原生 App。专为家长设计，帮助轻松记录和激励孩子每日良好行为习惯的养成，支持多孩子独立档案管理、自定义习惯项目、科学数据图表统计、个性化奖惩约定，以及军工级高可靠性的 WebDAV 云端数据同步。

---

## 🎨 视觉风格主轴：几何平衡与软萌卡通 (Geometric Balance)
*   **色调组合**：使用温馨的水蜜桃粉、柔和玫瑰色、天空蓝与暖象牙白背景搭配，营造出让家长感到舒心、让孩子感到亲切的童趣交互体验。
*   **特色插画**：在主看板积分及周历单元格中绘制了极具现代艺术感的手绘拟人“小红花”笑脸动画。
*   **触角精度**：所有可点按操作按钮均符合 Material Design 3 标准的 48dp 最小触控范围，按压带有饱滑的阻尼微动波纹水波效果。

---

## 🚀 核心功能模块划分

### 1. 👶 多宝贝独立档案管理
*   **多玩家机制**：支持添加多个不同孩子的独立成长记录，每个孩子拥有专属于自己的习惯定义、红花总计数、周历表现、历史图表以及奖惩约定。
*   **快速切换/管理**：顶端卡片中内嵌便捷的“切换”徽章。点击弹出专属底板：
    *   查看所有已添加宝贝；
    *   单击切换当前活动宝贝（无缝瞬间重载底层全部流动数据流）；
    *   可视化新增宝贝，创建完即刻自动注入默认初始化生活习惯项目；
    *   支持删除闲置的多余孩子档案。

### 2. 📝 表现项目（习惯）全功能自定义 (CURD)
*   **按需设项**：打破固定习惯限制，家长可在单孩子下添加、修改、彻底删除（或软删除）任何行为项目（如：早睡早起、准时进餐、认真写作业、体育运动）。
*   **简便排版**：提供侧边和底部的列表预览，方便一目了然浏览当前生效的多维好习惯。

### 3. 📅 核心每日表现周视图记录区 (Core Daily Grid)
*   **极简横向周历**：横轴为周一至周日及对应当前月份日期，纵轴为自定义好习惯项。
*   **小红花瞬时勾选**：轻轻点按单元格，完成时自动浮现绽放的小红花笑脸大徽章，再次轻按即可秒级清空，数据单日独立，多孩子数据彼此严格物理隔离。
*   **周选轮换**：支持一键切换上周、下周或者回到本周。

### 4. 📊 科学数据多维度统计 charts
*   **本周习惯完成率**：实时统计当前孩子本周各习惯项在 7 天之中的完成比例。
*   **月度红花日历看板**：支持自由变换要统计的“年份与月份”，高亮渲染当月每天累计获得的红花总量。

### 5. 📒 孩子奖惩规则约定
*   **红花奖励兑换**：自由设定表现换取对应实体奖励的制度（如：累计 20 个红花可换取一次去游乐园的机会）。
*   **行为纠正规章**：作为规范，柔性约束不文明举止，支持随手修改。

### 6. ☁️ WebDAV 工业级云端无缝同步
*   **高兼容性兼容**：支持全球任意 WebDAV 服务器协议（包含坚果云 Jianguoyun、Nextcloud、Owncloud 等）。
*   **三种科学操作模式**：
    1.  **智能双向同步**：自动拉取云端 JSON 备份与本地 SQLite 进行高级双向归并，如未产生冲突则合并两端所有的宝贝、习惯及每日红花打卡（首选模式，杜绝覆盖数据丢失）。
    2.  **备份上云**：一键将本地最新整套库编码为格式化 JSON 备份文件安全上传覆盖云端。
    3.  **覆盖恢复**：提供醒目的双重红线警示框。点击校验并彻底还原云端数据到本地（清空本地以便完全替代）。
*   **安全防御**：支持应用授权密码隐字模式展示。

## 🛠️ 项目技术栈剖析

*   **构建语言**：Kotlin + JDK 17
*   **界面框架**：Jetpack Compose (声明式纯 Kotlin 编码，彻底告别老旧 XML 布局)
*   **持久层**：SQLite + Room (实体、DAO、Repository 三层标准 MVVM 结构)
*   **网络框架**：OkHttp + Json Parser Moshi (进行云端 WebDAV 安全异步传输)
*   **生命周期/并发**：Kotlin Coroutines (协程并发) + Flow (冷温流实时高响应追踪渲染)
*   **版本建议**：支持 Android 8.0 (API 等级 26) 及以上的任何手机、折叠屏及平板电脑设备。

## 💡 使用步骤说明

1.  **添加孩子档案**：进入主界面，默认创建“小明”档案，点击档案头像旁的“切换”配置自定义新名称及奋斗目标。
2.  **规划起步习惯**：默认自动带入常态基础健康习惯（早起、作业、阅读等）。点击习惯栏的加号，可以加入您的家庭专属新习惯。
3.  **每日开花打卡**：孩子表现优异时，在对应的网格星期格子里轻点，小红花绽放！
4.  **配置云同步**：
    *   在最下方 WebDAV 同步卡片中输入您的 DAV 服务器路径、账号及应用令牌。
    *   点击“智能双向同步”，保障全家不同设备、多手机之间数据的一致。
